# Introduction #

Mugo is go editing program and GUI interface for GTP engine.
  * English and Japanese UI support.
  * supported formats:
    * read: sgf/ugf/ngf/gib
    * save: sgf
  * sgf collection support.
  * branch view.
  * print.
  * tabbed interface.
